# SINPF LCMS

A Legal Case Management System built with Next.js, designed for managing legal cases, filings, and reports.

## Tech Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Fonts**: Outfit, Public Sans, Space Mono

## Features

- User authentication (login, access-restricted pages)
- Dashboard with cases, filings, and reports
- Case form with tabs for information, fees, uploads, etc.
- Responsive UI components

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd sinpf-lcms
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Application

Start the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## Project Structure

- `app/` - Next.js app router pages and layouts
- `components/` - Reusable UI components
- `public/` - Static assets
